# Simple_Project2_API_Flask_ReactJS
```
Transfer data via http in Python Flask dan ReactJS with setInterval. 
Pada backend dilengkapi multithreading untuk menjalankan fungsi timestamp().  
```
# Cara Menjalankan Aplikasi 
```
backend:
  [+] python3 server.py
frontend:
  [+] npm start
```
